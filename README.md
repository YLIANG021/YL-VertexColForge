# YL VertexColForge

YL VertexColForge is a Blender add-on built for artists who want vertex color work to feel fast, direct, and production-ready.

Paint, bake, preview, randomize, mirror, blend, and reshape vertex colors without bouncing between scattered tools. It brings the everyday vertex color workflow into one focused panel, with practical controls for color channels, gradients, ambient occlusion, mesh transfer, and visual feedback.

## Highlights

- Vertex color painting tools with focused brush controls.
- Gradient workflows for UV and 3D color generation.
- Ambient occlusion, curvature, and gradient baking helpers.
- Channel operations for previewing, mirroring, layering, and random fills.
- Mesh, texture, and weight transfer utilities.
- Bundled preview asset for display workflows.

## Installation

1. Download the extension package or clone this repository.
2. In Blender, open `Edit > Preferences > Extensions`.
3. Install or load the add-on, then enable `YL VertexColForge`.
